# CIFAR-10 图像分类 AI 训练与推理工具
这是一个基于 PyTorch 构建的卷积神经网络 (CNN) 图像分类项目，专为 CIFAR-10 数据集设计。该程序采用交互式命令行菜单，将“模型训练”与“图像识别”无缝集成在单一脚本中，适合快速进行模型验证和本地图像批量分类。

# 核心功能
二合一工作流：通过终端交互菜单，一键切换“训练模式”与“识别模式”。

优化的 CNN 架构：内置自定义的 DeeperCNN，包含 3 层卷积、Batch Normalization、Dropout 以及自适应平均池化层。

智能断点续训：训练过程中每 10 轮自动保存 Checkpoint。若训练中断，下次启动时可自动加载权重、优化器状态和学习率调度器状态，无缝恢复进度。

动态学习率与早停机制：使用 CosineAnnealingLR 调整学习率，并设定了目标准确率（默认 80%），达标后自动停止训练以节省算力。

自动化数据增强：内置随机裁剪、色彩抖动、水平翻转和随机旋转，提升模型泛化能力。

开箱即用的批量推理：自动创建并扫描本地的图片文件夹，对输入的外部图像自动进行尺寸缩放和归一化处理后输出预测结果。

🚀 使用说明
1. 启动程序
在终端或命令行中运行脚本：

Bash
python main.py
程序会自动检测可用的硬件设备（CUDA/CPU）并展示主菜单：

Plaintext
========================================
  欢迎使用 CIFAR-10 “二合一”AI工具
========================================
...
请选择您的“工作模式”：
  (1) 训练模式 (继续学习/生成新大脑)
  (2) 识别模式 (扫描文件夹，获取识别结果)
  (3) 退出程序
2. 模式一：训练模型 (输入 1)
初次运行该模式会自动从网络下载 CIFAR-10 数据集到 ./data 目录下。

系统按照设定的最大轮数（1500 轮）开始训练。

当测试集准确率达到 80.0% 时，触发 Early Stopping 机制结束训练并保存最终模型。

如果在中途退出程序，重新输入 1 即可从最新的 Checkpoint 恢复训练。

3. 模式二：识别图像 (输入 2)
需确保当前路径下存在已训练好的模型权重文件（my_cifar10_brain_v4.pth）。

程序会自动检测或创建 ./predict_images 文件夹。

将你需要预测的图片（支持 .jpg, .jpeg, .png, .webp）放入该文件夹中。

在菜单中选择 2，程序将遍历文件夹并输出每张图片的预测类别。

支持识别的 10 个类别：
飞机、汽车、鸟类、猫、鹿、狗、青蛙、马、船、卡车

# 核心参数配置
如果您需要调整训练参数，可以直接修改 main.py 中的相关变量：

目标准确率：修改 YOUR_TARGET_ACCURACY = 80.0

最大训练轮数：修改 num_epochs = 1500

批次大小 (Batch Size)：在 DataLoader 中修改 batch_size=64

初始学习率：在 optim.SGD 中修改 lr=0.01

模型文件名称：修改 BRAIN_FILE = 'my_cifar10_brain_v4.pth'

# 注意事项
多进程支持：为了解决在某些操作系统上的兼容性问题，代码入口已强制设置为 spawn 启动方法。

图像尺寸要求：通过 predict_images 文件夹输入的图像不限制初始分辨率，程序在推理时会自动进行 Resize((32, 32)) 的缩放处理，但这可能会导致非正方形图片发生形变，从而轻微影响预测准确率。
